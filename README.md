# 🤖 AI Voice-Controlled Robotic Hand (Jarvis)

## 🚀 Overview
This project is an AI-powered robotic hand controlled using voice commands through a Jarvis-style assistant.

## 🎯 Features
- Voice control using AI assistant
- 5-finger robotic hand
- Python-based control system
- Future integration with ROS2 & Gazebo

## ⚙️ Technologies
- Python
- Machine Learning
- Robotics
- AI
